# LaTex OCR

## 这是什么？

这是基于开源项目[LaTex-OCR](https://github.com/lukas-blecher/LaTeX-OCR)搭建的**公式图片转LaTex数学公式**应用。算力和技术支持——[北京大学Linux俱乐部](https://lcpu.club/)。

**请勿上传非数学公式图片，请勿上传过大的文件！**

## 使用方法

1. 使用本地文件：拖拽或点击网页中的文件上传框，选择需要识别的公式图片，目前限制格式为jpg/png，上传后等待获得模型预测结果，结果会自动复制到剪贴板中。

2. 使用截图：利用截图工具截图后，直接在网页中粘贴，即可自动上传并识别。

可以在左侧文件框内选择文件查看之前的预测结果，服务器不会保留识别记录，切换页面或刷新网页将会导致历史记录消失。

实测准确率较高，对于复杂公式能够较好的完成识别，下面是一个成功的例子：
